COMO JOGAR
  1. Clique em "Iniciar";
  2. Jogador 1 clica no lugar onde deseja inserir um X;
  3. Jogador 2 clica no lugar onde deseja inserir uma bolinha;
  4. Clique em "Reiniciar" para reiniciar o jogo e zerar os pontos.
REGRAS
  • O ponto será marcado automatica para o jogador que completa uma linha;
  • Caso nenhum jogador complete uma linha, será dado como velha.
